# 述职评分系统

一个现代化的述职评分管理系统,支持评委打分、述职人员查看成绩和管理员系统配置。

## 功能特性

### 评委端
- 选择评分周期和述职人员
- 使用滑块拖动方式为每个题目打分
- 实时保存评分,可修改已提交的分数
- 支持批量保存所有评分

### 述职人员端
- 查看自己的评分详情
- 查看各评委的打分明细
- 查看各题目平均分和总平均分
- 支持按评分周期查询

### 管理员端
- **用户管理**: 灵活添加/删除评委和述职人员
- **题目管理**: 自定义评分题目,可设置分数范围和步长
- **周期管理**: 设置评分周期,支持多个周期管理
- **评分统计**: 查看所有述职人员的评分结果,自动计算平均分

## 核心功能

1. **灵活的用户配置**: 可以随时添加或删除评委和述职人员
2. **自定义评分题目**: 支持设置题目名称、最小分、最大分和步长
3. **滑块评分**: 通过拖动滑块选择分数,保留一位小数
4. **多人评分收集**: 收集所有评委的打分,计算平均分
5. **评分周期管理**: 支持创建多个评分周期,独立管理

## 技术栈

- **框架**: Next.js 14 (React 18)
- **语言**: TypeScript
- **样式**: Tailwind CSS
- **图标**: Lucide React

## 快速开始

### 安装依赖
```bash
npm install
```

### 启动开发服务器
```bash
npm run dev
```

访问 http://localhost:3000 即可使用系统

## 使用说明

### 首次使用
1. 在首页选择角色(评委端/述职人员/管理员)
2. 选择对应的用户身份
3. 点击"进入系统"

### 管理员配置流程
1. 进入管理端
2. 在"用户管理"中添加评委和述职人员
3. 在"题目管理"中设置评分题目和分数范围
4. 在"周期管理"中创建评分周期
5. 在"评分统计"中查看评分结果

### 评委打分流程
1. 选择评分周期
2. 选择述职人员
3. 拖动滑块为每个题目打分
4. 点击保存按钮提交评分

### 述职人员查看流程
1. 选择评分周期
2. 查看各题目的评委打分明细
3. 查看平均分和总分

## 默认数据

系统预置了一些示例数据:
- 管理员、2位评委、2位述职人员
- 3个评分题目(工作完成质量、团队协作能力、创新能力)
- 1个评分周期(2025年第一季度)

## 数据说明

- **评分范围**: 可自定义,默认 0-10 分
- **评分精度**: 保留一位小数
- **评分步长**: 可自定义,默认 0.1
- **平均分计算**: 所有评委有效评分的算术平均值
